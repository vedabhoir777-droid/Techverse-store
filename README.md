# 🛒 TechVerse Store

A modern e-commerce website built with React and Supabase that provides a smooth technology shopping experience.

## 🌐 Live Demo

🔗 Vercel Link: https://techverse-store.vercel.app/

## ✨ Features

- 🔐 Supabase Authentication (Login/Signup)
- ➕ Add Products
- 🗑️ Delete Products
- 🛍️ Shop Products
- 📂 Product Collections
- 🎨 Modern Responsive UI

## 🛠️ Tech Stack

- React.js
- JavaScript
- Supabase
- HTML & CSS
- Vite
